Artificial Intelligence Project – Prediction of logP

In this project, I leveraged my Python programming skills and focused on using key libraries like Scikit-learn to develop machine learning models. The primary goal was to predict the important chemical property, logP, based on various chemical descriptors. Specifically, I implemented the Random Forest Regressor, Gaussian Process Regressor, and Support Vector Regressor algorithms. Through rigorous experimentation and evaluation, I systematically assessed the performance of these three regressor models in predicting logP. All of this work is meticulously documented in a Jupyter Notebook, providing clear insights into the practical applications of machine learning, Python programming, and their relevance in addressing real-world challenges in the field of chemoinformatics.
